# 線性迴歸學習系統
**R 語言實作入門教材**

> 📖 **一個嘗試整理線性迴歸學習資源的小專案**

---

## 📚 關於這個專案

這是一個學習性質的專案，整理了一些線性迴歸的基本概念和 R 語言實作範例，希望能幫助初學者更容易入門。內容主要針對統計學、資料分析等相關課程的學習需求。

### 教材內容

這個專案包含：

- **互動式學習介面**：`index.html` - 可以在網頁中瀏覽所有教材
- **4 份 R 語言教材**：從基礎入門到簡單的分析範例
- **圖片資源**：一些用 R 生成的圖表範例

### 💭 建議學習順序

- **完全新手**：從「R快速上手指南」開始
- **有 R 基礎但不熟悉線性迴歸**：可以先看「理論與實作對照」
- **準備做專題或作業**：建議看完前面三份教材再看「完整案例分析」

每個人的需求不同，這只是一個參考建議。

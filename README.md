## 📚 教材系統簡介

本系統整合**理論知識**與 **R 語言實作**，提供從入門到進階的完整學習路徑。

### 兩大核心系統

#### 1. 互動式理論學習系統
- **檔案**：`線性迴歸學習系統.html`
- **特色**：網頁互動介面，涵蓋理論基礎、數學推導、統計推論
- **來源**：整合中國醫藥大學教材、Wooldridge 計量經濟學
- **適合**：理論學習、概念理解、數學推導

#### 2. R 語言實作教材系列 ⭐ NEW
- **4 份完整教材**：從快速上手到完整案例分析
- **特色**：程式碼可直接執行、逐步解說、實戰導向
- **適合**：動手實作、資料分析、研究報告

## ❓ 常見問題 FAQ

### Q1: 我沒學過統計，能用這套教材嗎？
**A:** 可以！從「R快速上手指南」開始，搭配理論學習系統的「快速入門路徑」（30分鐘），就能開始分析資料。

### Q2: 教材之間的順序重要嗎？
**A:** 重要。建議依照學習路徑的順序。如果跳過基礎直接看進階，可能會遇到很多未解釋的概念。

### Q3: 我可以只學 R 不學理論嗎？
**A:** 技術上可以，但不建議。理解理論才能：
- 判斷模型是否可靠
- 知道何時該用何種診斷
- 正確解讀結果

### Q4: 教材會持續更新嗎？
**A:** 會。我們會根據使用者反饋持續改進，並新增更多實戰案例。

### Q5: 可以用於商業用途嗎？
**A:** 教材僅供教育用途。商業使用請聯繫維護者。

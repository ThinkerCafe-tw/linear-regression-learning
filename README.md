## 常見問題 FAQ

### Q1: 為什麼需要線性迴歸？
**A:** 線性迴歸可以量化變數間的關係，進行預測，並檢定關係的顯著性。廣泛應用於社會科學、經濟學、生物統計等領域。

### Q2: R² 要多大才算好？
**A:** 取決於研究目的。預測應用需要高 R²（>0.8）；了解變數關係則重點在參數顯著性，R² 低於 0.2 仍可能有意義。

### Q3: 如果假設不滿足怎麼辦？
**A:**
- **非線性**：嘗試變數轉換（對數、平方）
- **異質變異數**：使用穩健標準誤或 WLS
- **非常態**：大樣本下 OLS 仍具漸近常態性

### Q4: OLS 與 MLE 有何差異？
**A:** 在常態性假設下兩者相同。MLE 需要分配假設，但可推廣至複雜模型；OLS 不需常態假設，計算簡單。

### Q5: 如何更新知識庫？
**A:** 直接編輯 `linear_regression_knowledge.json` 檔案，JSON 結構清晰，易於維護。

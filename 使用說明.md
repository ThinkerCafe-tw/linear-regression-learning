# 線性迴歸學習系統 - 使用說明

## 問題：無法載入 JSON 檔案

當您直接開啟 `線性迴歸學習系統.html` 時，瀏覽器的 CORS 安全政策會阻止載入外部 JSON 檔案。

## 解決方案（選擇其中一種）

### 方案 1：使用 Python 本地伺服器（推薦）

這是最簡單且最可靠的方法：

1. **開啟終端機（Terminal）**
   - Mac：按 `Command + 空白鍵` 搜尋「終端機」

2. **進入線性迴歸資料夾**
   ```bash
   cd "/Users/thinkercafe/Library/Mobile Documents/com~apple~CloudDocs/MEGA下載/RECORD/授課與演講/線性迴歸"
   ```

3. **啟動 Python 伺服器**
   ```bash
   python3 -m http.server 8000
   ```

4. **開啟瀏覽器**
   前往：http://localhost:8000/線性迴歸學習系統.html

5. **停止伺服器**
   按 `Ctrl + C`

### 方案 2：使用瀏覽器設定（Chrome）

1. **關閉所有 Chrome 視窗**

2. **以特殊模式啟動 Chrome**
   ```bash
   open -na "Google Chrome" --args --disable-web-security --user-data-dir="/tmp/chrome_dev"
   ```

3. **開啟 HTML 檔案**
   直接拖曳 `線性迴歸學習系統.html` 到 Chrome

⚠️ **注意**：此方法會關閉安全設定，僅用於本地測試，使用後請關閉此 Chrome 視窗。

### 方案 3：使用 VS Code Live Server

如果您使用 VS Code：

1. 安裝 **Live Server** 擴充套件
2. 在 VS Code 中開啟資料夾
3. 右鍵點擊 HTML 檔案
4. 選擇「Open with Live Server」

### 方案 4：上傳到網站空間

將以下檔案上傳到任何網站空間：
- `線性迴歸學習系統.html`
- `linear_regression_knowledge.json`

確保兩個檔案在同一目錄下。

## 常見問題

### Q: 為什麼會有 CORS 錯誤？
A: 瀏覽器的安全機制阻止本地 HTML 檔案載入其他本地檔案。使用本地伺服器可以繞過此限制。

### Q: Python 伺服器安全嗎？
A: 非常安全。它只在您的電腦上執行，不對外開放。

### Q: 我沒有 Python 怎麼辦？
A: Mac 預裝 Python，直接使用即可。Windows 用戶可使用 VS Code Live Server（方案 3）。

### Q: 可以分享給別人使用嗎？
A: 可以！只需將兩個檔案（HTML + JSON）一起給對方，對方也需要使用相同方法開啟。

## 建議使用方式

**日常使用**：方案 1（Python 伺服器）
**臨時測試**：方案 2（Chrome 無安全模式）
**開發使用**：方案 3（VS Code Live Server）
**分享使用**：方案 4（上傳網站）
